## 🧠 About The Project

> **Super Duper Waddle** is a Kanban-style task manager inspired by Trello — built from scratch to deeply understand how such systems actually work under the hood.

<table align="center">
<tr>
<td align="center" width="33%">🖱️<br/><b>UI Interaction</b><br/>Smooth, intuitive controls</td>
<td align="center" width="33%">🔄<br/><b>Drag & Drop Mechanics</b><br/>Built without shortcuts</td>
<td align="center" width="33%">⚙️<br/><b>Workflow Logic</b><br/>Real task-state management</td>

## 🚀 Getting Started

**1️⃣ Clone the repository**
```bash
git clone https://github.com/yourusername/super-duper-waddle.git
cd super-duper-waddle
```

**2️⃣ Install dependencies**
```bash
npm install
```

**3️⃣ Run the app**
```bash
npm start
```

<div align="center">

🎉 Your board should now be running at **`http://localhost:3000`**

</div>
