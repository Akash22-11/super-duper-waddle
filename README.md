## 🧠 About The Project

**Super Duper Waddle** is a Kanban-style task manager inspired by Trello — but built from scratch to deeply understand how such systems work.

This project focuses on:

* UI interaction
* Drag & drop mechanics
* Task workflow logic

---

## ✨ Features

<div align="center">

| 🚀 Feature      | 💡 Description          |
| --------------- | ----------------------- |
| 📋 Kanban Board | Organize tasks visually |
| 🔄 Drag & Drop  | Move tasks smoothly     |
| 📝 Task Manager | Add / Edit / Delete     |
| 💾 Persistence  | Save tasks locally      |
| 📱 Responsive   | Works on all devices    |

## 🚀 Getting Started

### 📥 Clone Repo

```bash
git clone https://github.com/yourusername/super-duper-waddle.git
cd super-duper-waddle
```

---

### 📦 Install Dependencies

```bash
npm install
```

---

### ▶️ Run App

```bash
npm start
```
